Тема приложения: Приложение для рецептов.

Описание: Приложение позволяет вести список рецептов с детальным просмотром, а также отслеживать их статус.

Функции приложения: 
•	Отображение списка рецептов
•	Просмотр подробной информации о выбранном рецепте
•	Просмотр описания и ингредиентов
•	Навигация между экранами (Jetpack Navigation / Compose)
