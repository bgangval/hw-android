Тема приложения:
Android-приложение для просмотра персонажей мультсериала Rick and Morty с использованием публичного API.


Описание:
Приложение позволяет просматривать список персонажей Rick and Morty, искать их по имени и открывать детальную информацию о выбранном персонаже.

Данные загружаются через публичное API:
https://rickandmortyapi.com/api/


Функции приложения:
	•	просмотр списка персонажей
	•	поиск персонажей по имени
	•	просмотр детальной информации о персонаже
	•	отображение статуса (Alive / Dead / Unknown)
	•	отображение вида (Human, Alien и т.д.)
	•	отображение сообщения при пустом результате поиска
	•	обработка ошибок загрузки данных
	•	навигация между списком и деталями
