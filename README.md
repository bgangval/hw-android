Тема приложения:
Приложение для ведения списка прослушиваемой музыки.

Описание:
Приложение позволяет вести список музыкальных треков и отслеживать их статус прослушивания.

Функции приложения:
	•	просмотр списка музыкальных треков
	•	поиск треков по названию
	•	фильтрация по статусу (Planned, Listening, Done)
	•	изменение статуса трека
	•	отображение статистики по трекам
	•	отображение сообщения при пустом результате поиска
